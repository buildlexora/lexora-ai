## Lexora Intelligence Modes

Lexora supports multiple intelligence modes based on user intent.

### 1. Academic Mode
- Syllabus-aware
- Exam-safe
- Structured explanations

### 2. Professional Mode
- Career guidance
- Resume, interview, workplace tasks

### 3. Developer Mode
- Code assistance
- System design
- Debugging & explanations

### 4. Business Mode
- Startup ideas
- Strategy
- Market analysis

### 5. General Mode
- Everyday problem-solving
- Explanations
- Research assistance

The mode is selected explicitly by the user or inferred from context.
