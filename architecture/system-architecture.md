# System Architecture ⚙️

## Core Flow
User Intent → Intent Interpreter → System Architect → 
Task Decomposition → Execution Engine → Output

## Engine Interaction Map
- Intent Interpreter feeds into System Architect
- System Architect feeds into Task Decomposition
- Task Decomposition feeds into Execution Engine
- Project State Memory persists across all engines
- Code Quality Guard validates before final output
- Iteration Engine loops back based on feedback
